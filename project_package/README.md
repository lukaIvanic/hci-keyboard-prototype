# Project Package Structure (Template)

This folder scaffolds the submission ZIP structure. Populate each subfolder with the final artifacts.

## Suggested structure
- `code/` - zipped source code.
- `data_raw/` - exported CSV/JSON logs.
- `analysis/` - analysis scripts, outputs, charts.
- `report/` - final report PDF and source files.
- `slides/` - presentation slides.
- `media/` - screenshots or demo video.
- `memorandum_template.md` - one-page project memorandum template.
