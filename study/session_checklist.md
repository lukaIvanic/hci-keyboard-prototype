# Session Checklist (Operator)

## Before the session
- Confirm participant code and condition label.
- Verify device and browser version.
- Confirm layout order and seed (if seeded).
- Confirm practice trial count and trials per layout.

## During the session
- Start the experiment and observe that practice trials appear first.
- Ensure the participant understands how to correct errors.
- Monitor for fatigue and allow short breaks if needed.

## After the session
- Export CSV and JSON.
- Record any anomalies or interruptions.
- Save files in the designated project package folders.
